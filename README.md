# 📘 Student Dashboard (Django Project)

A Student Dashboard System built using Django to manage student data, courses, fees, and overall dashboard insights in a modular way.

---

## 🚀 Project Overview

This project is structured using multiple Django apps to ensure scalability and maintainability. Each app is responsible for a specific functionality.

---

## 📂 Project Structure

- **core/** → Main project configuration  
- **accounts/** → User authentication and account management  
- **students/** → Student details (name, roll number, age, etc.)  
- **course/** → Course management  
- **fees/** → Fee records and transactions  
- **dashboard/** → Dashboard insights and summaries  

---

## ⚙️ Features

- Manage student information  
- Track courses  
- Fee management system  
- User authentication (Login/Signup)  
- Dashboard with summarized data  

---

## 🛠️ Tech Stack

- **Backend:** Django (Python)  
- **Frontend:** HTML, CSS  
- **Database:** SQLite
